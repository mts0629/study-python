# study-python

Study of Python

- `packages/` ... Tutorial for 3rd party packages
- `playground/` ... Small projects

- venv
- [mypy](https://github.com/python/mypy) (1.4.1)
- [pytest](https://github.com/pytest-dev/pytest) (7.4.0)
